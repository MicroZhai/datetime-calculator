# 时间计算器 V1.0 使用说明书

## 一、软件概述

**软件名称**：时间计算器（Time Calculator）

**软件版本**：V1.0

**开发语言**：HTML5 / CSS3 / JavaScript（纯前端）

**运行环境**：Chrome / Edge / Safari / Firefox 等现代浏览器，支持移动端和桌面端

**主要用途**：一款轻量级 PWA 日期时间计算工具，支持多段时间链式计算、分组管理、计算历史等功能。

---

## 二、安装与启动

### 移动端（推荐）
1. 在手机浏览器中打开应用网址
2. 点击浏览器菜单 →「添加到主屏幕」
3. 主屏幕出现「时间计算器」图标，点击即可使用

### 桌面端
1. 在浏览器中打开应用网址
2. 即可直接使用，无需安装

### 离线使用
首次加载后，应用会自动缓存所需资源，支持离线使用。

## 三、功能介绍

### 3.1 主界面

界面从上到下依次为：

**顶部栏**：
- 左侧：🕘 计算历史按钮
- 中间：应用标题
- 右侧：⚙ 设置按钮

**实时时钟**：显示当前日期和时间，每秒自动更新。

**分组选择器**：居中下拉框，可选择不同分组查看计算器。

**计算器卡片列表**：显示所有已保存的计算器，每张卡片包含：
- 计算器名称
- 开始时间、总时长、结束时间（三列布局）
- 展开/收起计算过程详情（竖线时间轴 + 时长小标签）
- 📌 置顶标识和跨天标识

**底部悬浮按钮**：
- ＋ 新建计算器

### 3.2 新建/编辑计算器

1. 点击底部 ＋ 按钮或卡片，打开编辑弹窗
2. **名称**：输入计算器名称，如"会议结束时间"
3. **开始时间**（只读）：显示时段 1 的开始时间
4. **「此刻」按钮**：一键将开始时间设为当前时间
5. **时段列表**：每个时段包含开始时间、时长（小时+分钟输入）、结束时间三字段
   - 修改任一字段，其余联动计算
   - ✕ 清除按钮可快速清空时长
   - 支持多时段，点击「+ 添加时段」新增
6. **历史时长**：自动记录常用时长值，点击快速填入
7. 点击「保存」完成

### 3.3 卡片操作

**点击卡片**：打开编辑弹窗

**点击 ⋮ 按钮**（卡片右上角）弹出菜单：
- 📌 置顶/取消置顶：重要计算器固定在最前
- ✏️ 重命名：编辑计算器
- 📋 复制计算详情：复制完整计算过程到剪贴板
- 📁 移动到分组：将计算器归类到指定分组
- 🗑️ 删除：删除该计算器

**点击总时长**（中间列）：快速进入编辑

**点击 ▼ 展开过程 / ▲ 收起过程**：展开或折叠每个时段的计算详情

### 3.4 计算历史

1. 点击顶部 🕘 图标，打开历史记录弹窗
2. 每条记录显示：计算器名称、开始/结束时间、完整计算过程、保存时间
3. **🔄 再用一次**：复制该记录的数据到新建编辑弹窗
4. **✕**：删除单条历史
5. **清空全部历史**：底部按钮，确认后清除所有历史记录
6. 点击弹窗外区域或关闭按钮关闭

历史记录自动保存，最多 50 条，超出后自动删除最旧记录。

### 3.5 设置面板

1. 点击顶部 ⚙ 图标，打开设置弹窗
2. **主题模式**：点击「切换」按钮，在浅色/深色/自动三种模式间循环
3. **管理分组**：
   - 输入分组名称，点击「添加」创建新分组
   - 分组旁 ✕ 按钮可删除分组（计算器自动移回"全部"）
   - 主界面的下拉选择器可快速切换分组

### 3.6 置顶功能

1. 右键菜单点击「📌 置顶」
2. 置顶的计算器排在最前，卡片左侧有强调线，左上角有 📌 图标
3. 再次点击取消置顶

### 3.7 数据存储

所有数据存储在浏览器 localStorage 中，包括：
- 计算器列表
- 计算历史（最多 50 条）
- 时长输入历史（最多 20 条）
- 主题设置
- 分组列表

## 五、使用示例

### 示例 1：计算会议结束时间

1. 新建计算器，命名"项目评审会"
2. 时段 1：开始时间设为 14:00，输入时长 90 分钟
3. 结束时间自动计算为 15:30
4. 点击保存，卡片显示结果

### 示例 2：多段烘焙时间计算

1. 新建计算器，命名"面包烘焙"
2. 时段 1：揉面 60 分钟
3. 时段 2：发酵 120 分钟
4. 时段 3：烘烤 40 分钟
5. 每个时段独立计算，最终显示总结束时间

### 示例 3：工作计划分组

1. 在设置中创建"工作""生活""学习"三个分组
2. 将工作相关计算器移动到"工作"分组
3. 通过下拉选择器切换分组查看

## 六、技术说明

- **开发语言**：HTML5 + CSS3 + JavaScript（ES6+）
- **框架**：无框架，纯原生实现，零外部依赖
- **响应式**：支持移动端和桌面端自适应
- **主题**：CSS 变量实现深色/浅色/自动切换
- **持久化**：localStorage 存储所有数据
- **离线**：Service Worker 网络优先缓存策略
- **存储限制**：受浏览器 localStorage 限制（通常 5-10MB）
